# Ипотечный калькулятор с историей расчётов и PDF-отчётами
Веб-сервис представляет собой полноценное приложение для расчёта параметров ипотеки с системой учётных записей пользователей и возможностью генерации отчётов.
Назначение и целевая аудитория
Основная задача: Автоматизация расчёта ежемесячных платежей, общей переплаты и других ключевых параметров ипотечного кредита. Сервис решает проблему сохранения истории расчётов и предоставления пользователю структурированного, готового к печати отчёта.

# Целевая аудитория:
Потенциальные заёмщики, оценивающие свои финансовые возможности.
Риэлторы и финансовые консультанты для предварительных расчётов с клиентами.
Студенты, изучающие финансовую математику или веб-разработку.

# Основные функции
Управление учётными записями:
Регистрация и аутентификация пользователей.
Защищённый доступ к персональной истории расчётов.
Расчёт ипотечных параметров:
Поддержка аннуитетной и дифференцированной схем платежей.
Расчёт на основе: стоимости недвижимости, первоначального взноса, процентной ставки и срока кредита.
Мгновенное получение результатов: ежемесячный платёж, общая сумма выплат, переплата.
Работа с историей:
Автоматическое сохранение каждого выполненного расчёта в базу данных.
Просмотр, фильтрация и управление историей (удаление записей).
Генерация PDF-отчётов (Интеграция с внешним API):
Автоматическое создание профессионально оформленных PDF-документов для любого сохранённого расчёта.
Интеграция со сторонним сервисом PDF.co для облачной генерации PDF.

# Архитектурные особенности и технологии
Backend: ASP.NET Core 8.0 (C#)
Архитектура: Многослойная (Controllers, Services, Models, Data)
База данных: SQLite с использованием Entity Framework Core (Code-First, миграции)
Аутентификация: Cookie-based аутентификация
Фронтенд: Razor Pages (встроенный UI), HTML/CSS
Внешняя интеграция: REST API клиент для взаимодействия с PDF.co (HTTP-запросы, JSON)

# Использование
Зарегистрируйте нового пользователя или войдите в систему.
На главной странице заполните параметры ипотеки и нажмите «Рассчитать».
Для просмотра истории перейдите по ссылке «История» или «Моя история расчётов».
В истории нажмите «Скачать PDF» рядом с любым расчётом, чтобы получить отчёт в формате PDF.

Для просмотра истории перейдите по ссылке «История» или «Моя история расчётов».

В истории нажмите «Скачать PDF» рядом с любым расчётом, чтобы получить отчёт в формате PDF.
